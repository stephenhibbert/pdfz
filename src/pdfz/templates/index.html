Remote (recommended)
Connect to the deployed PDFZ server. Add to your project's .mcp.json:
{ "mcpServers": { "pdfz": { "type": "url", "url": "https://pdfz-production.up.railway.app/mcp/", "headers": { "Authorization": "Bearer <token>" } } } }
  1. Copy the API token above and replace <token> in the JSON
  2. Add to your project's .mcp.json and restart Claude Code
  3. Three tools will be available:
    list_documents — browse all ingested PDFs
    get_document_toc — view a document's table of contents
    search_document_pages — ask questions about specific pages
Local (development)
Run the MCP server locally as a stdio subprocess:
{ "mcpServers": { "pdfz": { "command": "uv", "args": [ "run", "--directory", "/path/to/pdfz", "pdfz-mcp" ] } } }
  1. Update the path to your local pdfz directory
  2. Make sure ANTHROPIC_API_KEY is set in the pdfz project's .env
