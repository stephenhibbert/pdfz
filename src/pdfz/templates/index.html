{ "mcpServers": { "pdfz": { "command": "uv", "args": [ "run", "--directory", "/path/to/pdfz", "pdfz-mcp" ] } } }
  1. Add the JSON above to your project's .mcp.json file (update the path)
  2. Make sure ANTHROPIC_API_KEY is set in the pdfz project's .env
  3. Restart Claude Code — three tools will be available:
    list_documents — browse all ingested PDFs
    get_document_toc — view a document's table of contents
    search_document_pages — ask questions about specific pages
